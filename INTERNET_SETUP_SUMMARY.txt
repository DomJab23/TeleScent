## Quick Setup Instructions

### âš¡ Option 1 (Fastest)
```bash
# 1. Download ngrok
# 2. Run: ngrok http 5000
# 3. Copy the URL (e.g., https://abc123.ngrok.io)
# 4. Create frontend/.env:
echo "REACT_APP_API_URL=https://abc123.ngrok.io" > frontend/.env
# 5. npm run build in frontend folder
# 6. docker-compose up
# 7. Done! Access from phone at: https://abc123.ngrok.io
```

## Security Notes

ğŸ”’ **Your app is now public on the internet!**

1. **Update JWT_SECRET** - Change the default in backend/.env
2. **Use HTTPS** - All options above use HTTPS (you're secure)
3. **Whitelist origins** - Update ALLOWED_ORIGINS in backend/.env
4. **Rate limiting** - Backend includes this by default
5. **Monitor logs** - Check for suspicious activity: `docker logs -f telescent-app`

## Common Problems & Solutions

| Problem | Solution |
|---------|----------|
| "Can't connect from phone" | Check if using public URL (not localhost) |
| "Port already in use" | Change port in docker-compose.yml |
| "CORS error" | Add origin to ALLOWED_ORIGINS in .env |
| "API calls fail" | Ensure REACT_APP_API_URL is set in frontend/.env |
| "URL changes when restart" | Use Option 2 or 3 for stable URL |
